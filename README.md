## CPU-Shader

A framework allowing for quick and easy creation of shader like programs that run on the CPU. These CPU-Shaders are not as performant as there GPU counterparts however, they are much easier to debug, and easier to develop for many reasons most notably that memory is much easier to work with.

Every CPU-Shader is run for every pixel of the screen on the CPU and its final result is copied over to the GPU in order to display it.
